# ⚡ レンジ計算機

電子レンジのワット数変換ツール（PWA対応）

## 📱 特徴

- **シンプル操作**：ワット数をタップ、時間を入力、計算ボタンを押すだけ
- **オフライン対応**：一度開けばネット不要で使える
- **マルチデバイス**：Android / iOS / iPad で動作
- **ホーム画面追加**：アプリのように使える

## 🚀 使い方

### 1. ブラウザで開く

#### ローカルで試す場合
```bash
# シンプルなHTTPサーバーを起動
cd microwave-app
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開く

#### サーバーにアップロードする場合
すべてのファイルをWebサーバーにアップロードし、URLを開く

### 2. ホーム画面に追加（アプリ化）

#### Android（Chrome）
1. Chromeでページを開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. アイコンがホーム画面に追加される

#### iOS / iPad（Safari）
1. Safariでページを開く
2. 下部の共有ボタン📤をタップ
3. 「ホーム画面に追加」を選択
4. アイコンがホーム画面に追加される

### 3. 使い方

1. **もとのワット数**を選択（500W / 600W / 800W / 1500W）
2. **時間**を入力（分と秒）
3. **変換先のワット数**を選択
4. 「計算する」ボタンをタップ
5. 変換後の時間が表示される

## 🔧 カスタマイズ

### アイコンを変更したい場合
`icon-192.png` と `icon-512.png` を任意の画像に差し替える

### ワット数を変更したい場合
`index.html` の以下の部分を編集：
```html
<div class="watt-option" data-watt="500">500W</div>
```

### 色を変更したい場合
`index.html` の `<style>` 内のカラーコードを編集

## ⚙️ 技術仕様

- **計算式**：(元W × 元時間) ÷ 新W
- **端数処理**：1秒単位で切り上げ
- **対応ワット数**：500W / 600W / 800W / 1500W
- **エラーチェック**：
  - 時間が0の場合
  - 元Wと変換先Wが同じ場合

## 🐛 トラブルシューティング

### ホーム画面に追加できない
- Android：Chrome以外のブラウザでは対応していない可能性があります
- iOS：必ずSafariを使用してください（Chromeは非対応）

### オフラインで動かない
- 一度オンラインで開いて、すべてのリソースを読み込んでください

### 計算結果がおかしい
- ブラウザのキャッシュをクリアして再読み込みしてください
